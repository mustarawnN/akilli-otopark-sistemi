# akilli-otopark-sistemi
🚗 Akıllı Otopark Yönetim Sistemi (Full-Stack & IoT)

Bu proje, fiziksel sensör verileriyle (ESP32/Arduino) entegre çalışabilecek, ölçeklenebilir ve gerçek zamanlı bir "Akıllı Otopark Yönetim Sistemi"nin web altyapısını oluşturur. Önce Yazılım (Software-First) mimarisiyle geliştirilmekte olup, donanım entegrasyonu öncesi sağlam bir Backend ve Veri Tabanı temeli üzerine inşa edilmektedir.
