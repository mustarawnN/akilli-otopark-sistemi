Akıllı Otopark Sistemi (MusPark)
Fiziksel bir otopark maketi (ESP32 ve IR sensörler) ile tam entegre çalışabilen, React ve Node.js tabanlı kapsamlı bir akıllı otopark yönetim panelidir.

🚀 Temel Özellikler
İnteraktif U-Kroki: Otoparkın fiziksel dizilimine (U şeklinde) birebir uygun, anlık doluluk durumunu ve plakaları gösteren canlı harita.

VIP Abonelik Modülü: Belirli alanları (A1, A2) abonelere rezerve eden, aktif üyeliklerde çıkış ücretini 0₺ olarak hesaplayan ve süresi dolan abonelikleri otomatik tespit eden akıllı sistem.

Gelişmiş Finansal Raporlar: Saatlik yoğunluk, blok bazlı gelir, tarife dağılımı ve günlük/haftalık/aylık ciro analizleri.

Canlı Akış (Live Feed): Otoparka giren ve çıkan araçların hareketlerini anlık olarak takip edebilme.

Dinamik Fiyatlandırma: Yönetim paneli üzerinden saatlik ve günlük park tarifelerini anında güncelleyebilme.

Oto-Simülasyon Modu: Sistemi test etmek için rastgele plakalarla otomatik araç giriş/çıkışı sağlayan bot entegrasyonu.

Karanlık/Aydınlık Tema: Kullanıcı tercihine göre değişebilen modern arayüz tasarımı.

🛠️ Kullanılan Teknolojiler
Frontend: React, Tailwind CSS, Heroicons

Backend: Node.js, Express.js

Veritabanı: MS SQL Server (mssql)

Donanım (Planlanan): ESP32, IR Engel Algılama Sensörleri
